FUN3D HELP >5. L Systems > 5.4. Supported Rules and Syntax

List of supported rules and syntax for value field of rules property

Variables: All letters between A and T
Constants:
  1. >x Move Forward along object X axis
  2. <x Move Backward along object X axis
  3. >y Move Forward along object Y axis
  4. <y Move Backward along object Y axis
  5. >z Move Forward along object Z axis
  6. <z Move Backward along object Z axis
  7. +x Rotate Clockwise around X axis
  8. -x Rotate Counter-Clockwise around X axis
  9. +y Rotate Clockwise around Y axis
  10. -y Rotate Counter-Clockwise around Y axis
  11. +z Rotate Clockwise around Z axis
  12. -z Rotate Counter-Clockwise around Z axis
  13. [ Save Transformation
  14. ] Restore Transformation
Examples:
F>x+zF>x-zF>x-zF>x+zF
Procedures:
  1. Set F for Initial property then edit rules property
  2. Set F for Name property and F>x+zF>x-zF>x-zF>x+zF for value property
  3. Sometimes you must change number of iterations to see result
